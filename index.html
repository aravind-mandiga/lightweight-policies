Method Overview

MobileACT Architecture

  • Vision Compression: MobileNet-V3 Small (0.9M params) replaces ResNet-18 (11.2M params)
  • Transformer Reduction: 2-layer encoder + 1-layer decoder with 128-D hidden states
  • Parameter Efficiency: 95% reduction in transformer parameters
  • Total: 1.78M parameters (29Ã— reduction)

SSMPolicy Architecture

  • State-Space Models: Mamba-style selective SSM blocks for O(n) complexity
  • Factorized Embeddings: 4Ã— compression in parameter efficiency
  • Linear Complexity: Efficient sequence processing vs. O(nÂ²) attention
  • Total: 1.45M parameters (36Ã— reduction)
