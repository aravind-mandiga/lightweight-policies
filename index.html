MobileACT Architecture

Our first lightweight alternative targets aggressive parameter reduction while preserving ACT's fundamental design.

  • Vision Compression: MobileNet-V3 Small (0.9M params) replaces ResNet-18 (11.2M params)
  • Transformer Reduction: 2-layer encoder + 1-layer decoder with 128-D hidden states
  • Parameter Efficiency: 95% reduction in transformer parameters
  • Total: 1.78M parameters (29× reduction)

SSMPolicy Architecture

Our second approach leverages state-space models to achieve even greater efficiency with linear complexity.

  • State-Space Models: Mamba-style selective SSM blocks for O(n) complexity
  • Factorized Embeddings: 4× compression in parameter efficiency
  • Linear Complexity: Efficient sequence processing vs. O(n²) attention
  • Total: 1.45M parameters (36× reduction)

Architecture Overview

MobileACT Architecture

MobileACT preserves ACT's core conditional VAE framework while introducing lightweight innovations for vision processing and sequence modeling.

SSMPolicy Architecture

SSMPolicy demonstrates the first application of selective state-space models to chunked action prediction in robotic manipulation with linear computational complexity.
